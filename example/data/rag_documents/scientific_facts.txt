科学事实汇编

物理学基础事实：

1. 光速
光在真空中的速度约为 299,792,458 米/秒（约30万公里/秒）。这是宇宙中信息传播的最快速度。

2. 重力
重力是物体之间的吸引力。地球表面的重力加速度约为 9.8 m/s²。

3. 原子结构
原子由质子、中子和电子组成。质子和中子在原子核中，电子在核外运动。

4. 能量守恒
能量不能被创造或毁灭，只能从一种形式转换为另一种形式。

生物学基础事实：

1. 人体骨骼
成年人有 206 块骨头。婴儿出生时有约 270 块骨头，随着生长部分骨头融合。

2. DNA
DNA（脱氧核糖核酸）携带遗传信息，以双螺旋结构存在。

3. 细胞
细胞是生命的基本单位。人体由数万亿个细胞组成。

4. 光合作用
植物通过光合作用将光能转化为化学能，产生氧气和葡萄糖。

化学基础事实：

1. 水的化学式
水的化学式是 H₂O，由两个氢原子和一个氧原子组成。

2. 元素周期表
目前已知的元素有 118 种，其中 94 种是自然存在的。

3. 酸碱度
pH 值衡量溶液的酸碱性，范围从 0（强酸）到 14（强碱），7 为中性。

4. 化学反应
化学反应涉及原子重新排列形成新物质，但不会创造或毁灭原子。

天文学基础事实：

1. 太阳系
太阳系有 8 大行星：水星、金星、地球、火星、木星、土星、天王星、海王星。

2. 地球自转和公转
地球自转一周约 24 小时，绕太阳公转一周约 365.25 天。

3. 月球
月球是地球的天然卫星，距离地球平均约 384,400 公里。

4. 银河系
太阳系位于银河系中，银河系包含数千亿颗恒星。

医学基础事实：

1. 疫苗工作原理
疫苗通过激发免疫系统产生抗体，使身体在未来遇到真正病原体时能够快速反应。

2. 抗生素
抗生素只对细菌感染有效，对病毒感染无效。

3. 血型
人类血型分为 A、B、AB、O 型，以及 Rh 阳性或阴性。

4. 人体器官
人体有许多重要器官，包括心脏、肺、肝、肾、大脑等，每个都有特定功能。

地球科学事实：

1. 地球结构
地球由地壳、地幔、外核和内核组成。

2. 板块构造
地球表面由多个板块组成，它们缓慢移动导致地震和火山活动。

3. 大气层
地球大气层主要由氮气（78%）和氧气（21%）组成。

4. 水循环
水通过蒸发、凝结、降水和径流在地球表面循环。

数学基础事实：

1. 圆周率
π（pi）约等于 3.14159，是圆周长与直径的比值。

2. 毕达哥拉斯定理
在直角三角形中，斜边的平方等于两直角边的平方和。

3. 质数
质数是只能被 1 和自身整除的大于 1 的自然数。

4. 零的性质
任何数乘以零都等于零，但不能除以零。

重要科学原理：

1. 进化论
物种通过自然选择随时间演化，适应性强的个体更容易生存和繁殖。

2. 相对论
爱因斯坦的相对论描述了时间、空间、质量和能量之间的关系。

3. 量子力学
描述微观粒子行为的物理理论，与日常经验大不相同。

4. 热力学定律
描述能量转换和熵增的基本物理定律。

这些是经过科学验证的基础事实，被广泛接受并有大量证据支持。
